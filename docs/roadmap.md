# Incremental Roadmap

## Milestone 1: Product Foundation

- Monorepo structure
- Prisma schema
- Auth API skeleton
- Premium responsive frontend shell
- Admin and user dashboard previews
- Kanban, timeline and command palette surfaces

## Milestone 2: Real Authentication

- Register, login, refresh and logout flows wired to PostgreSQL
- Email verification token model
- Password reset flow
- Protected routes and RBAC guards

## Milestone 3: Project Core

- Workspaces, teams, projects, versions and milestones
- Kanban persistence
- Activity log and comments
- Socket.io board updates

## Milestone 4: Agile Planning

- Sprint planning
- Velocity, burndown and burnup
- Backlog grooming
- Epics, stories, features, tasks, bugs and hotfixes

## Milestone 5: Collaboration

- Notifications
- Internal chat
- Mentions
- Documents and previews

## Milestone 6: Intelligence and Exports

- AI service adapter
- CSV, Excel, PDF and JSON exports
- Risk detection and sprint generation workflows
